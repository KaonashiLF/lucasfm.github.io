Python é uma linguagem de programação orientada a objetos, mas que funciona muito bem e é muito utilziada como programação estruturada. Ela pode ser utilizada para criação de softwares, criação de websites (django), automação de processos, análises descritivas e preditivas de dados e até mesmo criação de dashboards.


Bibliotecas de Python:

  • Pandas: É uma biblioteca utilizada para trabalhar com análise e manipulação de dados.
  • Numpy: É uma biblioteca focada em operações matemáticas, mas que também pode ser utilizada para manipução de dados. Funciona muito bem com Pandas
  • Selenium: Biblioteca focada em automações utilizando web
  • Matplotlib: Também utilizada com dados, esta é uma biblioteca focada em amostragem dos dados, com visualização, animação e interação de gráficos
  • Django: Este na verdade é um framework para criação de websites responsivos.
